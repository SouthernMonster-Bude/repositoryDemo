主机名配置
hostname 查看主机名
hostname 临时名称 暂时修改
/etc/sysconfig/network 修改配置文件可以永久性修改主机名


ip地址配置
方法一：
setup
root输入setup命令进入交互式修改界面
minimal安装的系统没有setuptool工具可用
yum install setuptool ntsyssv system-config-securitylevel-tui
yum install system-config-network-tui system-config-keyboard
进行安装
方法二：
修改配置文件
vim /etc/sysconfig/network-scripts/ifconfig-eth0
service network restart
reboot
方法三：
ipconfig eth0 192.168.0.2 ## 临时ip

网络监听
yum install net-tools
netstat -nlpt 获取进程名 进程号 用户id
netstat -tnl 查看处于监听状态的TCP端口和连接


安装httpd
yum install httpd
服务启动在80端口
物理机可以在浏览器上访问虚拟机的ip
如果主机访问不了则确保双向ping通
1.修改防火墙设置：在Shell下输入命令 /sbin/iptables -I INPUT -p tcp --dport 80 -j ACCEPT
2.保存上面的设置：/etc/rc.d/init.d/iptables saved
3.重启防火墙：/etc/init.d/iptables restart
