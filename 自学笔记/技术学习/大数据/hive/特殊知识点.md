## hive sql

### Like

% 代表零个或多个字符(任意个字符)。

_ 代表一个字符

### RLIKE

RLIKE子句是Hive中这个功能的一个扩展，其可以通过Java的正则表达式这个更强大的语言来指定匹配条件。

```mysql
（1）查找以2开头薪水的员工信息
 select * from emp where sal LIKE '2%';
（2）查找第二个数值为2的薪水的员工信息
 select * from emp where sal LIKE '_2%';
（3）查找薪水中含有2的员工信息
 select * from emp where sal RLIKE '[2]';

```

正则规则

```
\	转义
^	表示一行的开头
$	匹配一行的结束
*	匹配上一个字符0次及以上
.	匹配一个任意字符
[]	表示匹配某个范围内任一一个字符

```
